#  AI Airbnb Agent using MCP

An AI-powered Airbnb search agent built with **PydanticAI**, **Model Context Protocol (MCP)**, and **vLLM**.

This project demonstrates how AI agents can interact with external tools through MCP servers to search Airbnb listings and retrieve detailed property information using an OpenAI-compatible API.

---

##  Features

- AI Agent built with PydanticAI
- Model Context Protocol (MCP) integration
- OpenAI Compatible API
- vLLM local inference server
- Airbnb search tool
- Listing details retrieval
- Tool Calling workflow
